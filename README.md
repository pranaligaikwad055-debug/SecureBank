# SecureBank
#  SecureBank Management System

A simple and secure Bank Management System developed using the C programming language.  
This project is designed for beginners to understand the concepts of banking operations, file handling, structures, and menu-driven programming in C.

---

# 📌 Project Overview

The SecureBank Management System allows users to perform basic banking operations through a console-based interface.  
Users can create accounts, login securely, check account balances, and transfer money between accounts.

This project demonstrates the practical implementation of:
- Structures in C
- Functions
- File Handling
- Conditional Statements
- Loops
- User Authentication System

---

#  Features

##  Secure Login System
- User authentication using username and password
- Simple secure access to banking operations

##  Account Creation
- Create new bank accounts
- Store customer information securely

##  Check Balance
- View current account balance anytime

##  Money Transfer
- Transfer money between accounts

##  File Handling
- Store and retrieve account data using files

##  Menu-Driven Program
- Easy-to-use console interface
